Instructions

Game Controls

The main game contols are right under the bar. To start the game and each new round you will have to select "Start Round". When you lose you will have to select "New Game" before starting a new round.

If you are feeling up for a challenge, select "hard mode". This will add another glass after each succesful round.

Game Options

You do not need to enter any game options to play the game, they are optional. If you wish to change the number of glasses in the sequence to remember you can change that in the game options. If you wish to change change the maximum number of rounds to win the game you can change that too.

This game is meant to be challenging but still fun. If you are having difficulty remembering the full sequence please feel free to lower the game options.

Gameplay

After you select "Start Round" get ready for the fun to begin!

After a very short delay the first glass will blink. You will need to select the correct glass. If you select the correct glass the first two glasses will blink in a sequence. You will need to select the correct glasses in the correct sequnce to pass the round.

The sequence will continue to grow until the max sequence is reached.

Each successful round costs $5. Be careful because you only came to the bar with a certain amount of money.

Bribery

Did you accidentally click the wrong glass? That's ok. If you have enough money you can choose to tip the bartender for the chance to try again.

Be careful if you choose to do this. The bartender will not always accept tips and the amount is different every time. You may get kicked out of the bar anyway!
